# How to download video from Youtube easily

Install [Docker](#)

Run [Youtube Dowload]() on Windows PowerShell or Linux Bash:

```bash
docker run --net host -v .:/app -w /app tmvdl/yt-dl:dlp-alpine yt-dlp "<url>"
```

Change `<url>` to Youtube Video URL.
