# Conda CLI Tips

## Create new environment

```bash
conda create -n venv python=3.10
```

## Activate environment

```bash
conda activate venv
```

## Install packages

```bash
conda install numpy pandas matplotlib
```

## List environments

```bash
conda env list
```

## Remove environment

```bash
conda env remove -n venv
```
